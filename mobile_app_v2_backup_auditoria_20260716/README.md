# KorzadiVPN Android

## Requisitos
- Android Studio Iguana o superior (para desarrollo local)
- JDK 17
- Android SDK 34 (API Level)

## Compilación
Este proyecto utiliza GitHub Actions para la compilación automática.
Para compilar localmente:
`./gradlew assembleDebug`

## Instalación
`adb install app/build/outputs/apk/debug/app-debug.apk`
