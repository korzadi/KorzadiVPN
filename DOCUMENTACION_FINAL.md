# KorzadiVPN - Guía de Construcción y Preparación

## Backend
El backend está validado y preparado para producción.

## Clientes
- **Android:** Estructura de `mobile_app_v2` verificada. Requiere un entorno con JDK 17+ y Android SDK.
- **iOS/Linux/Windows:** Estructuras base creadas en sus respectivos directorios (`/ios`, `/linux`, `/windows`). Requieren implementación de la lógica de conexión WireGuard y cliente API REST.
